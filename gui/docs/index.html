WQX 工具箱

目前支持的工具：

  • GVBASIC 模拟器

配置文件

配置文件都位于本程序可执行文件所在的目录下，以下几个配置文件可由用户自行修改，然后点击菜单 <程序> -> <重新加载配置文件>，配置文件的改动就能生效。

  • machines.yaml   配置不同机型的 PEEK/POKE 地址等数据。
  • config.yaml   配置 GVBASIC 编辑器字体、模拟器屏幕颜色等信息。
  • styles/*.xml   GVBASIC 编辑器的文字颜色配置，支持配置多种颜色主题，并在 config.yaml 中切换主题。

如有问题或者建议，请前往 https://github.com/arucil/wqxtools 发起 issue。
